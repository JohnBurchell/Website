C++\SDL Platformer

A Small platformer written in C++ and using SDL. Not quite a finished project but very fun none the less. Easily one of the most enjoyable projects I've done as there were lots of new areas to cover.

In particular, I very much enjoyed learning how to implement collision detection and the common pitfalls this entails. Working with different timesteps and juggling physics updates was a great eye opener during this project. I also implemented some ideas from Mat Buckland's Programming Game AI By Example.
